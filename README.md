## 📖 Sobre o Projeto

Este projeto foi desenvolvido como um estudo prático de desenvolvimento de jogos utilizando Python e Pygame.

O objetivo foi recriar a experiência clássica do Flappy Bird, preservando sua essência enquanto adicionava algumas funcionalidades extras para fins educacionais e experimentação.

⚠️ Este projeto foi criado exclusivamente para fins educacionais.

Todos os direitos relacionados ao jogo original pertencem aos seus respectivos detentores.

## 📥 Instalação

### 1. Clone o repositório

```bash
git clone https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git
```

### 2. Entre na pasta do projeto

```bash
cd SEU-REPOSITORIO
```

### 3. Instale o Pygame

```bash
pip install pygame
```

ou

```bash
python -m pip install pygame
```

---

## ▶️ Como Executar

Dentro da pasta do projeto execute:

```bash
python main.py
```

Caso utilize múltiplas versões do Python:

```bash
python3 main.py
```
